# TIC TAC TOE (BOARD VS PLAYER)
A TTL Based TIC TAC TOE Game using Atgmega as EEPROM and FILP FLOP . Also Resistor used to store the value. This game played by board and player, where either player or the board gives the first move and the game must go on until its going to Draw or Lose or Win.

## Project Summary
This project is a custom electronics build that combines digital logic, memory, timing, and embedded control ideas. The design uses a set of flip-flops, registers, a timer IC, EEPROM, and an ATmega-style microcontroller platform with an external oscillator.A digital hardware implementation of the classic Tic-Tac-Toe game, inspired by a TTL-based design and later adapted with EEPROM and an ATmega-based control concept.

## Why This Project Matters
This work is a useful learning project for:

- digital logic design
- state machine implementation
- memory and EEPROM usage
- timing circuits
- embedded control concepts
- hardware prototyping

## Main Hardware Components
- 9 flip-flops for state storage and game logic control
- 1 timer IC for timing and sequencing
- registers for temporary data handling
- EEPROM for non-volatile storage
- ATmega-based control concept similar to an Arduino Uno-compatible IC
- oscillator for system clock generation

## Inspiration and Reference
The original concept was inspired by the TTL-based Tic-Tac-Toe project available at:

- https://github.com/DoctorWkt/TTL_TicTacToe

This repository documents a modified interpretation of that idea using EEPROM and ATmega-based control concepts.
